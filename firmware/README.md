# 펌웨어 저장소 (.hex 파일)

웹 브라우저에서 아두이노로 직접 코드를 업로드(플래싱)하기 위해서는 미리 컴파일된 바이너리 파일(`.hex`)이 필요합니다.

## 방법
1. PC의 아두이노 IDE에서 작성하신 `smartfarm.ino` 파일을 엽니다.
2. 상단 메뉴에서 **[스케치] -> [컴파일된 바이너리 내보내기]** 를 클릭합니다.
3. 스케치 폴더에 생성된 `.hex` 파일(보통 부트로더가 포함되지 않은 일반 .hex 파일)을 복사합니다.
4. 이 폴더(`firmware/`) 안에 `smartfarm.ino.hex` 라는 이름으로 붙여넣어 덮어씌웁니다.

이후 웹 화면에서 **[펌웨어 설치]** 버튼을 누르면 이 폴더에 있는 `.hex` 파일을 다운로드하여 아두이노에 업로드하게 됩니다!
